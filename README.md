# 个人作品集网站

## 如何修改内容

你的所有内容都在 `config.json` 里，直接在 GitHub 网页上编辑即可：

👉 **[点这里编辑 config.json](https://github.com/JiaYang66/portfolio2/edit/main/config.json)**

改完点 **Commit changes**，等 30 秒刷新网站就生效了。

## 添加照片

1. 进入 `photos/` 文件夹，按分类创建子文件夹
2. 上传照片（建议 JPG，每张不超过 500KB）
3. 在 `config.json` 的 `photos` 里填上文件名

## 添加视频

1. 把视频上传到 B站（bilibili.com）
2. 复制视频的 BV 号
3. 在 `config.json` 的 `videos` 里添加一条记录
